Classes

Class

A factory of class constructors that use js prototypal inheritance.

debug

The return of g3.debug when called as function is an object for debugging purposes.

Error

A custom error class.

headless

The return of g3.headless is a supplementary object which forms a root parent defining class with the help of the class factory g3.Class.

Highlight

A wrapper class for different highlight projects.

hybrid

The return of g3.hybrid is a supplementary object which forms a root parent defining class with the help of the class factory g3.Class.

HybridTemplate

measure

Returns an object that contains utility methods and properties that calculate the dimensions and position of an element, window, document or viewport or the value and unit in a string.
