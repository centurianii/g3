Tutorial: Highlight example class

Highlight example class

g3.Highlight example class

We implement this class as an example of the hybrid class system.

It abstracts three libraries and the use of the class follows the switchboardapproach; user supplies different object-values to the same methodinit()` and the system responds accordingly by loading some files and applying modifications on nodes.

A test page is built: an evaluator page filled with code tabs and stripped off the highlighting library that normaly is attached to the evaluator (see evaluator generic because in this case we actually want to test highlighting.
