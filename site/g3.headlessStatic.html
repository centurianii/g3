headlessStatic

g3. headlessStatic

Source:
  • g3headlessStatic.js, line 3
Version:
  • 0.1
Author:
  • https:/github.com/centurianii
Copyright:
  • MIT licence

The return of g3.headlessStatic is a supplementary object which becomes a mixin of the final defining class with the help of the class factory g3.Class.

It provides the static members under the context of the class that g3.Class will try to build, i.e. g3[myClass], or g3.myClass and for that reason the passed argument should be a string of the name of that class: 

g3.myClass = 
     g3.Class(
        g3.headless(myClass), 
        g3.headlessStatic(myClass),
        {
           STATIC: {...}, 
           constructor: function (...){...},
           ...
        }
     );

For inheritance see g3.hybrid.

It allows/blocks instance storage for this class at static property g3[myClass].instances.

g3.headlessStatic.STATIC.add

Source:
  • g3headlessStatic.js, line 231

This property will become static method of g3[myClass], g3[myClass].add().

It is called during instance construction and adds this instance to static array g3[myClass].instances with records of the form: {id: g3[myClass].instance.name, hybrid: obj}.

If the object already exists, it is not stored and an error is thrown, see g3.headless.constructor.
